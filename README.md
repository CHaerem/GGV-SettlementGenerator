# GGV-SettlementGenerator

## Overview

GGV-SettlementGenerator is a web application that automates the process of extracting donation information from PDF files generated by "Gi Gaven Videre AS". The application runs entirely in your browser - no server or Google account required.

## Live Application

**[Use the application here](https://chaerem.github.io/GGV-SettlementGenerator/)**

## Features

- **Client-side processing**: All PDF processing happens in your browser - your files are never uploaded to a server
- **Automatic text extraction**: Uses PDF.js to extract text from PDFs
- **OCR support**: Falls back to Tesseract.js OCR for scanned PDFs
- **Data validation**: Verifies extracted totals match calculated sums
- **CSV export**: Download extracted data as a CSV file for use in spreadsheets
- **Modern UI**: Clean, responsive interface with drag-and-drop support

## How to Use

1. Visit the [application page](https://chaerem.github.io/GGV-SettlementGenerator/)
2. Drag and drop a PDF file onto the upload area, or click to select a file
3. Wait for the processing to complete
4. Review the extracted data in the results table
5. Click "Last ned CSV" to download the data as a CSV file

## Extracted Data

The application extracts the following information from GGV settlement PDFs:

- **Organizations**: Names of charitable organizations
- **Amounts**: Donation amounts for each organization (in NOK)
- **Number of gifts**: How many gifts were given to each organization
- **Percentage**: Percentage of total gifts for each organization
- **Recipient company**: The company receiving the settlement
- **Total sum**: The total amount in the settlement
- **Cost contribution**: The cost contribution to Gi Gaven Videre

## Technology Stack

- **HTML5/CSS3**: Modern, responsive user interface
- **JavaScript (ES6+)**: Client-side application logic
- **[PDF.js](https://mozilla.github.io/pdf.js/)**: PDF text extraction
- **[Tesseract.js](https://tesseract.projectnaptha.com/)**: Optical Character Recognition (OCR)
- **GitHub Pages**: Static hosting

## Local Development

To run the application locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/CHaerem/GGV-SettlementGenerator.git
   cd GGV-SettlementGenerator
   ```

2. Serve the files using a local web server. For example, with Python:
   ```bash
   python -m http.server 8000
   ```

3. Open `http://localhost:8000` in your browser

## Deploying to GitHub Pages

1. Go to your repository settings on GitHub
2. Navigate to "Pages" in the sidebar
3. Under "Source", select "Deploy from a branch"
4. Choose the `main` branch and `/` (root) folder
5. Click "Save"

The site will be available at `https://[username].github.io/GGV-SettlementGenerator/`

## Privacy

This application processes all data locally in your browser. No PDF files or extracted data are ever sent to any server. Your files remain completely private.
